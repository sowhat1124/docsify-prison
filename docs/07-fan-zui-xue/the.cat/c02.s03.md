此一理論的代表人物是**佛洛伊德**，也就是探討做夢，著作「夢的解析」那位牛人。他認為人有與生俱來的犯罪潛意識，也就是「**本我**」本能上的這些衝動、野蠻、滿足自利益等行為，而這些只能靠天的學習與養成「**超我**」來控制它們，如果後天的控制不良就會產生犯罪。因此，每個人都有從事犯罪或偏差行為的本能。

### 壹、心理分析理論 / 一、理論內容 / （一）人格動力：

人在做行為時，是受到人格動力的影響。人格動力是一種驅使我們行動的力量，而這種趨力來自於下列這三個部分：

(Aicar)
1. **意識**：指個人經由感覺、知覺、思考、記憶對個人身心狀態與環境中人事物的綜合認識。
2. **下意識**：在現實當下無法察覺，可能是過去之經驗或記憶，經由特殊的方式牽引至意識當中。
3. **潛意識**：藉由壓抑而使其隱藏在意識之下，即生理之趨力和慾望、感情、恐懼等經驗，例如：性或攻擊。

(Cat)
1. **意識**：指個人經由感覺、思考、記憶對於這環境中的事物的認識，這些是可以直接體會到的。
2. **下意識**：在現實當下無法察覺，可能是過去之經驗或記憶，經由特殊的方式牽引至意識當中。

### 壹、心理分析理論 / 一、理論內容 / （二）人格結構：

1. **本我**：是人格最原始的部分，在人類出生時就存在的部分，代表野蠻、邪惡之衝動，要求立即滿足。使每個人具備從事犯罪的本能。受「快樂原則」所支配。
2. **自我**：人格中最理性、實際的部分，*受到現實之引導而做出的行為*，是根據外在規範來考慮行為時何者可以做，負責協調本我和超我間的衝突。受「現實原則」所支配。
3. **超我**：代表著個人潛在良心及人格道德部分，由社會規範以及雙親教育的價值觀於子女心中內化而成，受「理想原則」所支配。

### 壹、心理分析理論 / 一、理論內容 / （三）人格發展：

主要成長是在 6 歲前。

1. **口腔期**：約 0-2 歲，以吸吮、餵食、愛撫來獲得基本需求，若不足則會有挫折感及剝奪感產生。
2. **肛門期**：約 2-3 歲，藉由排泄訓練逐漸產生超我，若過分強調超我之訓練，可能會導致日後變成固執或虐待狂，反之則骯髒、草率。
3. **性器崇拜期**：約 3-6 歲，因為下意識而產生戀親情節，但最後會產生罪惡感而發展超我。有個更專業的名詞叫做「俄狄普斯期」。
4. **潛伏期**：約 6 歲後開始觀察外界之事物，人格中的超我發展快速，與異性間的關係較為疏離。
5. **兩性期**：約 12-13 歲進入青春期，喜歡參加兩性活動，開始有職業以及婚姻規劃，性心理發展即告成熟。

### 壹、心理分析理論 / 二、犯罪原因：

（因為我們是考犯罪學，考試請多加著墨這個部分。）

（一）從**人格動力**面觀之：心理分析學者認為處於精神層面中的潛意識會驅動我們的行為，甚至我們無法察覺。這樣的觀念讓犯罪學者們可以解釋許多不合常理的犯罪行為。

（二）從**人格結構**面觀之：本我控制不好，會因為人類原始的欲望和本能而造成暴力犯罪。自我負責調節本我和超我，如果自我發展不健全，容易形成反社會人格。超我的過度發展會將本我過度的壓抑，形成所謂的超現實人格而具有精神病或是偏執狂的特徵。

（三）從**人格發展**面觀之：人格形成的各個時期，若有得到滿足，也沒有過度的訓練，人格就可以得到健全的發展，不容易形成犯罪。

### 壹、心理分析理論 / 二、犯罪原因 / 犯罪原因許春金老師分類法：

（一）犯罪是一種**神經性行為**：由於某些無意識不為社會接受之趨力，而有強烈的罪惡感，因此有接受懲罰之衝動以減輕罪惡感。

（二）犯罪是**缺陷超我**的結果：犯罪者從小的社會化過程就很差，容易屈服在原「始趨力之下，因此產生犯罪，或是超我訓練太超過而成為激情犯。

（三）犯罪是**替代性滿足**的象徵：大多是幼兒時期需求被剝奪或未能滿足，而這些需求則在長大後藉由犯罪來滿足。例如：肛門期：虐待狂，口腔期：酗酒、吸毒。

### 貳、人格特質理論 / 一、理論內容：

學者**歐波特**反對心理分析理論的潛意和本能的觀點，他認為犯罪的發生是人格特質所導致，而人格的成長與改變是受到當下的經歷以及對於未來的希望 所影響的。人格特質理論認為**病態的人格特質**容易導致犯罪行為之發生。病態人格常見的特性有自我中心、不成熟、攻擊性、侵略性……等。

（一）人格的意義：  
所謂人格是指個人對人、對己對事物等各方面適應時，於其行為上所顯示的獨特性。

（二）人格的特質：  
1. 個人特質：指個人所獨有，由個人在遺傳與生活的環境下所形成。
2. 共同特質：由同一種族、文化、生活方式的人所共同擁有。

### 貳、人格特質理論 / 三、心理變態性格類型

> 情緒高昂型、誇張型、無力型、欠缺自信型、抑鬱型、爆發型、無情型、缺乏意志型、狂信型、心情易變型


> （**反社會人格類型**、**犯罪人格類型**）：學者——施耐德

（一）**情緒高昂型**：活潑、好動、樂觀、善于交談、笑口常開，但缺乏持續性與調和性，過度興奮即易產生爭吵、自傲、說謊與衝動等，本型若與無情型、爆發型、意志薄弱型、自我顯示型等複合時，易于發生搶劫、恐嚇、傷害之類的暴力犯罪或詐欺罪。

（二）**誇張型**：虛榮心與自我顯示慾均甚強烈，其行為多半是虛偽的，常誇張說謊。患者易患詐欺、偽造文書等罪行。

（三）**無力型**：患者常有頭痛、頭重、失眠、嘔吐等神經性症狀，且由此導致注意散漫、疲勞亢進、工作效率降低等狀態：此型與犯罪的關係較小。

（四）**欠缺自信型**：對於自己具有不確實感、不滿足感，因而缺乏自信，常拘泥于瑣碎事物而不能安心;此種人亦易于演成強迫性心理症或妄想狂患者，易患殺人、放火、傷害等罪。

（五）**抑鬱型**：患者以抑鬱性情為主，持厭世的人生觀，心情沈重，表情冷淡、猜疑心重、不快活、敏感，經常困居在家，不想外出，其與犯罪的關係亦較小。

（六）**爆發型**：易因輕微的刺激而激動，毫不顧慮地訴諸暴力，倾向于暴行、狂言、怒語、破壞等外在行為。患者易患暴行、傷害、殺人、妨害公務、破「壞器物等暴力罪行。

（七）**無情型**：缺乏同情、良心與羞恥心，而且有冷淡、殘酷與無恥的性格。患者易致財產侵犯罪、暴力犯罪與性犯罪等罪行。

（八）**缺乏意志型**：缺乏意志的持續力與行為的抑制力，對一切外力影響均無法反抗，常依他人暗示行事，受別人左右，無法下定決心。患者易犯竊盜、詐欺等財產罪，而犯殺人、傷害等暴行者較少。

（九）**狂信型**：行為常受強烈的感情思想所支配，常以個人特權、名譽、地位、「財產、利益等為其內涵。患者易于輪為殺人、傷害與興訟狂者。

（十）**心情易變型**：具有週期性的暴力或衝動行為，乃是由于心理緊張所激起，故易有衝動性的放火、偷竊、殺人等犯罪行為。

人之所以會犯罪都是因為道德發展水準低落造成的，因為對於社會道德規範的理解有誤差，表現出來的行為就常常不見容於社會，造成犯罪。

### 一、道德發展理論 / （一）皮亞傑

皮亞傑之研究認為道德發展是經由無律、他律、自律三個發展時期。

1. **無律時期**0-4歲：以單純的神經感應為主要行為，自我中心，並不十分了解規範，行為無法評價。
2. **他律時期**4-8歲：慢慢的如果兒童犯錯會被施以處罰，此期間行為的準則以服從權威，避免處罰為主。
3. **自律時期**8-12歲：10歲以後對於事理的判斷慢慢成熟，漸漸富於彈性，懂得自我管理。這時如果沒有正常發展仍停留在無律或他律階段就容易違反社會規範，形成犯罪行為。

### 一、道德發展理論 / （二）柯伯爾

> :memo: Recite:
> _____
> 6 個階段 / 前傳統階段 / 傳統階段 / 後傳統階段
> 避罰服從、相對功利、順從和諧、遵守法規、社會契約、公平正義


柯伯爾沿襲皮亞傑之理論，認為人們在成長的過程中經歷各種，每個階段對於對與錯的道德問題各。

1. 理論架構：  
分為**6 個階段**，其中第 1 階段及第 2 階段稱為**前傳統階段**，約由 0-10 歲之期間，第 3 及第 4 階段稱為**傳統階段**，約 10-15 歲期間，第 5 及第 6 階段稱為**後傳統階段**，約 15-20 歲之間，但並非指 20 歲者皆可達到後傳統階段。犯罪人多停留在前傳統階段。

   第 1 階段：**避罰服從**取向一個體關注行為的直接後果與自身的利害關係。例如，如果一個人由於某個行為而受到了懲罰，此行為被認為在道德上是錯誤的。

   第 2 階段：**相對功利**取向-個體做事情會想著「這件事對我有何益處」的想法，將正確的行為定義為對自己最有利的行為。只關注自己是否得到更多的利益，關心他人不是基於忠誠或內在的尊重。

   第 3 階段：**順從和諧**取向-扮演社會角色，個體關注其他人贊成或反對的態度，保持與周圍社會角色的和諧一致。他們努力要做一個「好孩子」，實現這些期待。

   第 4 階段：**遵守法規**取向-重要的是遵守法律和社會規範，維持社會有效運轉非常重要，認為社會的要求勝過個人的要求。如果有人確實觸犯了法律，那就是不道德的。

   第 5 階段：**社會契約**取向認為個體應持有自己的觀點和主張。因此，法律被看作是一種社會契約。法律應該達到「給最多的人帶來最大的利益」。民主政治顯然是基於第五階段的道德推理。

   第 6 階段：**公平正義**取向-道德推理是基於普世價值進行抽象推理。它超越了第四階段，認為只有在基於正義的情況下，法律才是有效的，不義的法律就不必服從。它也超越了第五階段，認為社會契約並非義務的道德行為之本質，會出現正義變成多餘之物的情況。

2. 對犯罪之解釋：  
   第3階段同前皮亞傑之自律時期是道德發展之關鍵點 ，若不能得到重要他人之肯定接，會影響其認知和行為發展，而更 失價值感和認同感，道德發展水準會停留在較低的階段。因此唯有*發展到高道德層次才能夠杜絕犯罪*在個人身上發生。

3. 評論：  
   研究對象只有男生，有人批評大男人主義。男女在道德判斷上有許多不同之處，不可據此解釋女性思維。

### 一、行為主義理論（制約）：

行為理論起步於20世紀初的美國，認為心理學研究是可以被觀察和測量統計的，人類行為是**學習**而來的，人類行為是**先天基因+後天環境**影響之結果。

### 一、行為主義理論（制約） / （一）代表人物：

> :memo: Recite:
> _____
> 桑貓、巴狗、史老鼠


1. **桑代克**：19世紀末美國學者，利用*貓走迷宮*實驗提出試誤學習習理論。
2. **巴普洛夫**：俄國生物學家，利用*狗與食物和鈴錯的連結*做實驗，發現制約與反射的關係。此反射是天生的，固定反射被激發的行為（流口水）。
3. **史基納**：美國行為學大師做了「史基納箱」的實驗，利用老鼠和拉桿子掉食物間的關係，提出了制約學習的概念。這邊的反射的動作是經過學習而強化。壓桿子有食物吃其壓桿子的動作頻率就會增加，其他沒有食吃的動作頻率就會減少。

### 一、行為主義理論（制約） / （二）理論主張內涵：

1. **人性白紙論**：認為「人和動物之行為是對外界刺激的反應，沒有任何思考判斷」，人性以是後天環境養成的。
2. *探討行為本身*：犯罪之行為受到痛苦（懲罰）與快樂（獎賞）所控制，所以應從行為本身來看未來的變化。
3. *行為是直接學習而來*：暴力犯罪主要是對於生活環境直接學習的結果，例如：從小被家暴，長大後易對自己家人家暴。
4. *增強與懲罰作用*：史基納提出的操作理論，認為有機體和環境互動而造成行為增加的過程稱為增強，反之若與環境互動後造成行為之減少，此過程即為懲罰作用。其中又可分為正面和負面，正面代表增加刺激產生反應，負面代表減少刺激產生反應。

因此增強與懲罰分為下列4種：

1. **正面增強**：在做完某個行為後，增加對象喜歡的刺激（獎勵），使得該行為的出現頻率增加。例如：放學後馬上寫完功課可以得到100塊。
2. **負面增強**：在做完某個行為後，減少（移除）對象厭惡的刺激，使得該行為的出現頻率增加。例如：每天如果可以早起，就不會挨打。
3. **正面懲罰**：在做完某個行為後，增加對象厭惡的刺激，使得該行為的出現頻率減少。例如：對於性侵犯只要有性衝動就使用電擊。
4. **負面懲罰**：在做完某個行為後，減少對象喜歡的刺激（獎勵），使得該行為的出現頻率減少。例如：玩樂後不收玩具，玩具會被沒收。

> 增強、懲罰：行為的出現頻率  
> 正面、負面：增加或減少刺激

### 一、行為主義理論（制約） / （三）對於犯罪預防之啟示：

1. *使用正面增強*：藉由家庭或是學校對於少年之行為多給予獎勵、認同，藉由正面增強之功效，使少年習慣於社會規範於日常生活當中。
2. *避免負面增強*：青少年為解決心中的煩惱、壓力，常常尋求不良友伴或是不良幫派的支持及安慰甚至藉由吸毒等方式紓壓，也容易受到幫派副文化影響而犯罪。因此要避免青少年使用負面增強的方式來排解憂慮。

（一）代表人物：塔德提出的模仿理論是針對布羅梭犯罪生物學理論而起。塔德認為犯罪者學習犯罪和其他行為一樣，都要經過一段的學徒生涯。在逐漸學習、訓練、模仿中形成他們的身分。

（二）理論內容：三種法則  

1. *距離法則*：人們模仿的對象，與自己越靠近，則行為模仿的強度則越大。在都市中，此種模仿最常見，但改變也快，稱之為「**時尚**」。在農村模仿較不常見，改變也慢，稱之為「**慣行**」。塔德認為犯罪*起之於時尚，而後變成慢行*。
2. *上行下效法則*：自歷史的觀點來看，許多行為都是從上層社會開始在流傳到世井小民。因此許多犯罪從都市發生再被鄉村所模仿。
3. *取代法則*：模仿有取代性，舊的時尚會被新的時尚所取代。

（三）貢獻：在19世紀塔德的理論有其時代的意義，即使至今日現代的犯罪學習理論，都源自於塔德的觀念。例如：*蘇哲蘭的差別接觸理論*。

### 三、社會學習理論 / （三）激起暴力的原因：

1. **教導**教唆：透過教導使之內化成為自己的行為模式，時機成熟就會使用暴力。
2. **厭惡**教唆：有點類似*負面增強*的概念，利用暴力來擺脫心理討厭的壓力挫折或情緒低落。
3. **楷模**教唆：當親近之人使用暴力卻沒有得到制裁，甚至得到尊重與獎賞，則觀察者也會因此仿效。
4. **妄想**教唆：逃避現實處於一種幻想之中，可能由於被害妄想而先發制人使用暴力。
5. **正面引誘**：預測暴力行為可能會得到獎賞或助益等正面效果，就會使用暴力來達成。

### 三、社會學習理論 / （四）持續使用暴力之原因：

1. **外在增強**：如果使用暴力總是得到有利的結果，行為就會一直出現。
2. **替代性增強**：記取前人的教訓，作為借鏡，而不必完全靠自己去體驗。看到別人的行為導致酬賞，本身會傾向做出相同的行為，而看到別人受罰，則會減低這種行為傾向。
3. **自我增強**：主要是為了維護自我價值。個體使用自發性的行為來獲得滿足和自我價值，並避免良心的譴責。也就是如果暴力行為可以提升自己的自尊或內在，那麼就會持續該行為，並使用*中立化技術*將責任歸罪於他人。

### 伍、心理病態性格 / 一、定義：

心理病態性格又稱為**社會病態人格**、**反社會人格**，指的是性格發展異常而**自我煩擾**並**且具有擾亂社會秩序之行為**。德國學者*施耐德之定義：「其人格較平均偏畸形，自身常煩惱於自身的異常性或社會為他的異常性所困擾。」*

### 伍、心理病態性格 / 二、特徵：

心理變態人格是真正的**核心犯罪者**，但並非有這些特徵都是心理變態性格，有些心理變態者的也沒有下列這些特徵，但*擁有越多特徵就越有可能是心理變態者*。

1. 無法與他人建立**忠誠**且持久的關係。
2. 不能從經驗中學到**教訓**。
3. 無**動機**的反社會行為。
4. 過度追求**刺激**。
5. 只為了**眼前**享樂，不考慮未來。
6. **無悔意**，無罪惡感。
7. 主要**情感**反應匮乏。
8. 極度**自我**中心。
9. 低**挫折**容忍力。
10. 慣於**玩弄**他人。

### 伍、心理病態性格 / 三、形成因素：

1. **生理**因素：
   1. *生理上的反應遲鈍*：在生理或情緒上反應遲鈍。
   2. *基因上的遺傳*：腦波緩慢、XYY超級染色體。
   3. *腦部異常*：腦部成長過程中受到傷害或是長腦瘤壓迫到神經。
2. **家庭**因素：在兒童早期受到父母不當對待，拒絕，不良教養。
3. **社會**文化因素：社會解組的社會，混亂動盪的社會。

### 伍、心理病態性格 / 四、類型：

1. **情緒高昂型**：經常陷入爽快的情緒中，看似樂觀但不負責任，不能信任。
2. **抑鬱型**：沉悶、自卑，常容易自責。厭世懷疑的人生觀，不易犯罪易自殺。
3. **缺乏自信型**：易屈服，缺乏自尊心，過於敏感，易生強迫性精神官能症。
4. **狂信型**：思想固執，會為自己所愛奮不顧身，常為政治犯或宗教狂熱分子。
5. **誇張型**：愛說大話，愛引人注意，虛榮心過強，易犯詐欺罪。
6. **心情易變型**：心情起伏大，缺乏耐性，會因鬱悶而出現不可預料之行為。
7. **爆發型**，對於刺激的反應不能平衡，常因瑣碎之事而暴怒。
8. **無情型**：缺乏同情、憐憫、良知、後悔、名譽、道德，最危險的類型。
9. **缺乏意志型**：卻乏持續性與獨立性意志力，容易受到誘惑。
10. **無力型**：具有習慣性神經質、神經衰弱等精神病現象，與犯罪較無關。

### 伍、心理病態性格 / 五、治療方式：

1. 培養互信基礎：因為幼年經驗導致無法與他人互動，矯正人員應先建立起良好互動，再逐步進行輔導。
2. 深植道德觀念：因為欠缺罪惡感，感受力低。因此需要灌輸其道德觀念。
3. 使用社區處遇：透過社區處遇等轉向處置，以環境療法來治療。

## 陸、犯罪心理學之貢獻與缺失：

> 還好

一、對犯罪防治貢獻：

1. **深入探討**犯罪行為之原因。
2. 協助犯罪**偵查**。
3. 提供法官**量刑**參考。
4. 促成矯正管理與**教化**工作之產生。
5. 可以運用在犯罪**預防實務**。

> 1, 2, 5 為萬用答案。

二、缺失：

1. 心理現象難以**預測**。
2. 研究有其界線及**範圍**過窄。
3. 研究結果不容易得到**驗證**。
4. 研究**角度**不符合刑法對於犯罪之評價。

## 柒、犯罪生物學及心理學對刑事政策之影響：

吹起**醫療模式**（詳監獄學）的風潮。

1. 使用**調查**分類方式來分別處遇。
2. **心理**治療。
3. **假釋**制度的使用。
4. 重視**更生**保護。
